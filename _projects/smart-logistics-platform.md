## Project Overview

An intelligent platform optimizing last-mile delivery operations through AI-driven route planning and real-time fleet tracking.

### Key Features

*   **Route Optimization:** AI algorithms to calculate the most efficient delivery routes.
*   **Real-Time Tracking:** GPS-based tracking of delivery vehicles for dispatchers and customers.
*   **Driver App:** Mobile application for drivers to manage deliveries and navigation.
*   **Proof of Delivery:** Digital signature and photo capture for delivery confirmation.

### Technologies Used

*   **Mobile:** React Native
*   **Backend:** Go (Golang)
*   **Containerization:** Docker
*   **Mapping:** Google Maps API / Mapbox
