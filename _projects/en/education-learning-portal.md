## Project Overview

A comprehensive LMS (Learning Management System) designed to facilitate online education with engaging content and progress monitoring.

### Key Features

*   **Course Management:** Tools for instructors to create and organize course content.
*   **Video Streaming:** High-quality video playback with adaptive bitrate streaming.
*   **Interactive Quizzes:** Assessment tools to test student understanding.
*   **Progress Tracking:** Detailed analytics on student performance and course completion.

### Technologies Used

*   **Frontend:** Angular
*   **Backend:** Python (Django)
*   **Cloud Infrastructure:** AWS (S3, CloudFront, EC2)
