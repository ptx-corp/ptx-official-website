## ภาพรวมโครงการ

แพลตฟอร์มที่ปรับขนาดได้สำหรับการตรวจสอบอุปกรณ์อุตสาหกรรมและเซ็นเซอร์สิ่งแวดล้อม ช่วยให้สามารถบำรุงรักษาเชิงคาดการณ์และเพิ่มประสิทธิภาพการดำเนินงาน

### คุณสมบัติหลัก

*   **การรวบรวมข้อมูลแบบเรียลไทม์:** การรับข้อมูลความถี่สูงจากเซ็นเซอร์นับพันตัว
*   **ระบบแจ้งเตือน:** การแจ้งเตือนที่ปรับแต่งได้สำหรับการละเมิดเกณฑ์และความผิดปกติ
*   **การแสดงภาพข้อมูล:** แดชบอร์ดสำหรับการแสดงภาพแนวโน้มเซ็นเซอร์และข้อมูลในอดีต
*   **การจัดการอุปกรณ์:** การกำหนดค่าระยะไกลและการอัปเดตเฟิร์มแวร์สำหรับอุปกรณ์ IoT

### เทคโนโลยีที่ใช้

*   **Backend:** Node.js
*   **Messaging:** MQTT Broker (Mosquitto/HiveMQ)
*   **Time-Series Database:** InfluxDB
*   **Visualization:** Grafana / Custom React Dashboard
