## ภาพรวมโครงการ

แพลตฟอร์มอัจฉริยะที่เพิ่มประสิทธิภาพการดำเนินงานจัดส่งระยะสุดท้ายผ่านการวางแผนเส้นทางด้วย AI และการติดตามยานพาหนะแบบเรียลไทม์

### คุณสมบัติหลัก

*   **การเพิ่มประสิทธิภาพเส้นทาง:** อัลกอริทึม AI เพื่อคำนวณเส้นทางการจัดส่งที่มีประสิทธิภาพที่สุด
*   **การติดตามแบบเรียลไทม์:** การติดตามยานพาหนะจัดส่งด้วย GPS สำหรับผู้จัดส่งและลูกค้า
*   **แอปคนขับ:** แอปพลิเคชันมือถือสำหรับคนขับเพื่อจัดการการจัดส่งและการนำทาง
*   **หลักฐานการจัดส่ง:** ลายเซ็นดิจิทัลและการถ่ายภาพเพื่อยืนยันการจัดส่ง

### เทคโนโลยีที่ใช้

*   **Mobile:** React Native
*   **Backend:** Go (Golang)
*   **Containerization:** Docker
*   **Mapping:** Google Maps API / Mapbox
