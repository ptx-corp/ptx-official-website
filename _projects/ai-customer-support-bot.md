## Project Overview

An advanced NLP-powered chatbot capable of handling complex customer queries and executing actions.

### Key Features

*   **Natural Language Understanding:** Accurately interprets user intent and sentiment.
*   **Automated Workflows:** Handles returns, order tracking, and FAQs automatically.
*   **Human Handoff:** Seamlessly transfers complex issues to human agents.

### Technologies Used

*   **AI/ML:** Python, TensorFlow, NLTK
*   **Frontend:** React widget
*   **Backend:** FastAPI
