## Project Overview

A secure and efficient system for managing patient data, scheduling appointments, and maintaining electronic medical records (EMR).

### Key Features

*   **Patient Portal:** Allows patients to book appointments, view medical history, and communicate with doctors.
*   **Doctor Dashboard:** streamlined interface for managing schedules and patient records.
*   **Telemedicine Integration:** Built-in video conferencing for remote consultations.
*   **HIPAA Compliance:** Ensures data security and privacy standards are met.

### Technologies Used

*   **Frontend:** Next.js, TypeScript
*   **Backend:** Node.js
*   **Database:** MongoDB
