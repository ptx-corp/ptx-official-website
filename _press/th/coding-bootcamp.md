# PT X เปิดตัวค่ายฝึกเขียนโปรแกรมฟรีสำหรับเยาวชนที่ขาดโอกาส

**กรุงเทพฯ, ประเทศไทย – 15 สิงหาคม 2024** – ในฐานะส่วนหนึ่งของความมุ่งมั่นต่อความรับผิดชอบต่อสังคม PT X ได้เปิดตัวค่ายฝึกเขียนโปรแกรมฟรีที่ออกแบบมาเพื่อมอบการศึกษาด้านการเขียนโปรแกรมคุณภาพสูงให้กับเยาวชนที่ขาดโอกาส

รุ่นแรกประกอบด้วยนักเรียน 100 คนจากชุมชนที่ขาดแคลน ซึ่งจะเข้ารับการฝึกอบรมเข้มข้นเป็นเวลา 12 สัปดาห์ ครอบคลุมพื้นฐานการพัฒนาเว็บ JavaScript และ React

"เราเชื่อว่าพรสวรรค์มีการกระจายอย่างเท่าเทียมกัน แต่โอกาสนั้นไม่ใช่" หัวหน้าฝ่ายบุคคลของ PT X กล่าว "ผ่านค่ายฝึกนี้ เราหวังว่าจะเปิดประตูให้กับเยาวชนที่อาจไม่มีโอกาสเข้าถึงอาชีพในด้านเทคโนโลยี"
